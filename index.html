🔥 Conectare Firebase
Introdu datele din proiectul tău Firebase pentru sincronizare live între toți participanții.
📋 Cum obții datele (2 minute, gratuit):
1. Mergi pe console.firebase.google.com → New Project
2. Build → Realtime Database → Create Database → Test mode
3. Setari proiect ⚙️ → General → copiezi apiKey si projectId
4. Din Realtime Database copiezi URL-ul (databaseURL)
